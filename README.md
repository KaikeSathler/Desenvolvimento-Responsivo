## Criando uma aplicação em Vue
```sh
$ npm create vue@latest
```
## Iniciando o projeto
```sh
$ cd <your-project-name>
$ npm install
$ npm run dev
```
> Note: É necesssario ter o `Node` Intalado, [Node.js](https://nodejs.org/pt).
