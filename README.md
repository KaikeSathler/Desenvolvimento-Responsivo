# vue-project

Este projeto web foi desenvolvido com base na disciplina Desenvolvimento Responsivo e Otimização WEB.

## Criando uma aplicação em Vue
```sh
$ npm create vue@latest
```
## Iniciando o projeto
```sh
$ cd <your-project-name>
$ npm install
$ npm run dev
```
